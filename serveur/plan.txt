Initialiser le projet

Configurer le serveur
Créer un fichier server.js avec un serveur Express de base, autoriser les requêtes depuis React (CORS), et accepter les requêtes JSON.

Créer les routes principales

POST /schedule : quand le frontend envoie un planning

GET /tasks : l’ESP32 vient chercher les tâches à exécuter

POST /ack (optionnel) : l’ESP32 confirme qu’il a nourri Mimi

Gérer les données
Pour commencer, stocker les tâches dans un fichier .json. Plus tard, tu pourras utiliser une vraie base de données si besoin.

Connecter l’ESP32
L’ESP32 appelle GET /tasks toutes les X minutes. Le serveur renvoie les actions prévues s’il y en a.

Ajouter de la sécurité
Authentifier le client et l'L’ESP32.

Tester
Tester les requêtes avec Postman. Vérifier que le frontend envoie bien les plannings et que l’ESP32 reçoit les bons ordres.
